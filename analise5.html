Riemann sums

The application returns a representation of Riemann sums for a polynomial function f of two variables (x, y) with a degree less or equal than two. The user should input the values of the polynomial coefficients a, b, c, d, e, and g, and the rectangular domain of the representation [xi, xs] x [yi, ys].

Use the horizontal bar to specify the number of parcels in the Riemann sum.
