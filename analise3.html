Gradient

The application returns the representation of the gradient vector of a polynomial function f of two variables (x, y) with a degree less than three. The user should input the polynomial coefficients a, b, c, d, e, g, and the coordinates of the point where the gradient vector will be calculated, (w, z).

Note that the pink line corresponds to the level curve with the level equal to f(w, z), and the green line represents the gradient vector.”
